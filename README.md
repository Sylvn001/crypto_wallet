<strong>Use DEV TASK for install project</strong>
``` 
  rails dev:setup  
```

OR.....
<br/>

<strong> Install depencencies </strong>
``` 
  bundle install
```

<strong> Create Database (SQLITE) </strong>
``` 
  rails db:create
```

<strong> Run migrations </strong>
``` 
  rails db:migrate
```

<strong> Run Seeds </strong>
``` 
  rails db:seed
```
